# Sources et Outils Utilisés dans le Projet Scoreboard Water-Polo

## Technologies et Frameworks

### Frontend
- **Tailwind CSS** (v3.x)
  - Source : https://cdn.tailwindcss.com
  - Utilisé pour le design responsive et moderne de l'interface
  - Documentation : https://tailwindcss.com/docs

- **Font Awesome** (v5.15.3)
  - Source : https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.3/css/all.min.css
  - Utilisé pour les icônes et éléments visuels
  - Documentation : https://fontawesome.com/v5/docs

### Polices
- **Oswald**
  - Source : https://fonts.googleapis.com/css2?family=Oswald:wght@400;500;600;700&display=swap
  - Police principale de l'application
  - Documentation : https://fonts.google.com/specimen/Oswald

- **Montserrat**
  - Source : https://fonts.googleapis.com/css2?family=Montserrat:wght@700&family=Roboto&display=swap
  - Police secondaire pour certains éléments
  - Documentation : https://fonts.google.com/specimen/Montserrat

- **Roboto**
  - Source : https://fonts.googleapis.com/css2?family=Montserrat:wght@700&family=Roboto&display=swap
  - Police de secours (fallback)
  - Documentation : https://fonts.google.com/specimen/Roboto
